# Election Analysis
This project analyzes vote counts in the form of a CSV to determine the winner of an election. The results are placed into a text document and include: voter turnout, vote total and percentage by county, candidate totals and percentages, and the winner of the election.

This works well for small elections and can handle any number of counties or candidates. Thought the data must be clean and cannot contain any mispellings. The winner in calculated using the popular vote.
